# Sweets日和

## サイト概要
### サイトテーマ
コンビニスイーツからホテルスイーツまで幅広い価格帯のスイーツ紹介したり、気になった商品はいいねで保存し見返すことができるレビューサイト

### テーマを選んだ理由
元々甘いものが好きでよく買っていましたが、いく場所や買う商品が定番化してしまっているので、他の人の投稿を見ることで新しい情報を得ることが出来ると考え、このテーマにしました。

### ターゲットユーザ
スイーツが好きな人

### 主な利用シーン
スイーツの情報が欲しい時、スイーツ情報を投稿する時
